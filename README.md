# ValentineWin32Application

Windows application created for my gf to celebrate Valentine`s Day

This application will show animation:
- 6 hearts with different colors will appear on screen
- They will start rotating
- After rotation they will combine into one huge Love letter

Application is written in win32 api, because my gf`s pc is very slow and low

# ValentineWin32Application

Приложение для Windows, созданное для моей девушки в честь Дня святого Валентина.

Это приложение показывает анимацию:
- На экране появляются 6 сердец разных цветов
- Они начинают вращаться
- После вращения они объединяются в одно огромное любовное письмо

Приложение написано на win32 api, потому что компьютер моей девушки слабее микроволновки
